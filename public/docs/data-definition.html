データ定義

PetroScope は public/data 配下の整形済み JSON を読むだけの静的アプリです。

円サイズ

oil-consumption-YYYY.json の metricName = oil_consumption を使用し、面積比例の円で描画します。

調達構成

oil-origin-mix-YYYY.json の domesticSharePct と importPartners を合計 100% として表示します。

データ品質

complete / partial / missing の 3 段階で品質を管理します。

国コード

すべて ISO 3166-1 alpha-3 を主キーに採用しています。

境界ポリシー

Natural Earth 由来の境界をそのまま使用します。アプリ独自の政治判定は行いません。
